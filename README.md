java-vs-scala
=============
To run the java code (requires Java 8)
...
cd java-test
mvn clean package
java -jar target/java-test-0.0.1-SNAPSHOT.jar
...

To run the scala code
...
cd scala-test
mvn clean package
java -jar target/scala-test-0.0.1-SNAPSHOT.jar
...

Open a browser to localhost:8080 to see the output.
